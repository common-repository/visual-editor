== Description ==

Improve your use of Gutenberg with the Visual Editor. Building your site will be even easier 

You can easily:

- View the outline of the blocks to better understand.
- Change the color of the guidelines, toolbar,sidebar and mouver
- View block names
- Direct remove button on toolbar
- Direct duplicate button on toolbar
- Mouver change position
- Auto open sidebar when click a block
- PreDesigned Layout with native Gutenberg Blocks!

Extra!

New! Visual Templates and Layout library made with native gutenberg blocks! 

The best intuitive visual editor!
